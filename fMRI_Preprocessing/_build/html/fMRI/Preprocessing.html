fMRI Preprocessing#

fMRIPrep#

For complete instructions, please visit the fMRIPrep Documentation.

Preparation Steps#

Applying for a FreeSurfer License

fMRIPrep uses FreeSurfer tools, which require a License to run.

Getting a Singularity Image

Spartan uses Apptainer (or previously Singularity) to run external applications (such as fMRIPrep), which requires a Singularity Image. fmriprep-23.2.1.sif is already available on Spartan in the AND Lab project directory. If a newer version is needed, you can obtain one by running the following commands line by line in the terminal:

module load GCC/11.3.0
module load Apptainer/1.2.3
export APPTAINER_TMPDIR=/tmp/<YOUR_USERNAME>/tmp
export APPTAINER_CACHEDIR=/tmp/<YOUR_USERNAME>/cache
mkdir -p $APPTAINER_TMPDIR $APPTAINER_CACHEDIR
apptainer build /data/gpfs/projects/punim2033/fmriprep-<VERSION>.sif docker://nipreps/fmriprep:<VERSION>

Adding an IntendedFor Field

For fieldmaps, you need to add an IntendedFor field to your fieldmap JSON sidecar files. This tells fMRIPrep which functional runs each fieldmap should be applied to. See the explanation of IntendedFor, or refer to the example below.

Main Commands#

apptainer exec --cleanenv \
    -B /path/to/input_dir:/data \
    -B /path/to/output_dir:/out \
    -B /path/to/work:/work \
    /path/to/fmriprep-<VERSION>.sif \
    fmriprep /data /out participant \
    --participant-label <SUBJECT_ID> \

Example Script

#!/bin/bash

#SBATCH --account=punim2033
#SBATCH --ntasks=1
#SBATCH --cpus-per-task=32
#SBATCH --nodes=1
#SBATCH --array=1
#SBATCH --job-name='fmriprep'
#SBATCH --time=0-20:00:00
#SBATCH -o /data/gpfs/projects/punim2033/safety_learning/slurm_output/fmriprep/slurm.%j.%N.out
#SBATCH -e /data/gpfs/projects/punim2033/safety_learning/slurm_output/fmriprep/slurm.%j.%N.err

# load modules
module purge
module load GCC/11.3.0
module load OpenMPI/4.1.4
module load minc-toolkit/1.9.18.2
module load Apptainer/1.2.3

# Suffix for directory naming
suffix="safety_fmri_prep"

# BIDS root directory
share_dir=/data/gpfs/projects/punim2033/safety_learning
bids_dir=${share_dir}/safety_fmri_bids

# extract sub id from array and change format
export sub=${SLURM_ARRAY_TASK_ID}

if [[ ${#sub} == 1 ]]
then
	export sub=0$sub
fi

# Place outside BIDS root directory, this allows rerun with minimal reprocessing
work_dir=${share_dir}/work/${suffix}
mkdir -p ${work_dir}
out_dir=$share_dir/${suffix}
mkdir -p ${out_dir}

# fmriprep
apptainer exec --cleanenv \
	-B ${bids_dir}:/data \
	-B ${out_dir}:/out \
	-B ${work_dir}:/work \
    -B ./license.txt:/license.txt \
	./fmriprep-23.2.1.sif \
	fmriprep /data /out participant \
    --participant-label ${sub} \
    --nthreads 16 \
    --fs-license-file /license.txt \
	-w /work \
	--skip_bids_validation \
	--output-spaces anat func MNI152NLin2009cAsym:res-native \
    --ignore slicetiming

my-job-stats -a -n -s
